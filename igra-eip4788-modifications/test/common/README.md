# Common Testing Files

This directory contains files shared between both testing approaches (reth-based and hardhat-based).

## Contents

- **`contracts/`** - Solidity wrapper contracts (BeaconRootWrapper.sol, RandaoGetterWrapper.sol)
- **`package.json`** - Common npm dependencies (Hardhat, ethers.js, etc.)
- **`artifacts/`** - Compiled contract artifacts (generated by Hardhat)
- **`cache/`** - Hardhat compilation cache

## Usage

Both `reth-testing/` and `hardhat-testing/` directories reference these common files through relative paths in their configurations.

To compile contracts:
```bash
cd test/common
npm install
npm run compile
```
